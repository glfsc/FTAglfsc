# 知识抽取功能 - 快速上手指南

## 🎯 一句话介绍

上传故障文档（PDF/TXT/DOCX），AI 自动提取故障知识三元组，一键生成故障树。

---

## 📝 3 分钟快速开始

## 🔧 常用操作

### 场景 1: 测试单个文件

```python
from app.services.knowledge_extraction_service import KnowledgeExtractionService

# 创建服务
service = KnowledgeExtractionService()

# 提取文件
result = service.extract_and_save("my_document.pdf")

print(f"✅ 提取到 {len(result['triplets'])} 个三元组")
print(f"📁 输出文件：{result['output_path']}")
```

**查看结果**: 打开 `data/output/my_document_triplets.json`

### 场景 2: 调用 API（Postman/Curl）

```bash
# 1. 上传文件
curl -X POST http://localhost:8000/api/v1/fault-tree/upload \
  -F "file=@test.pdf"

# 返回：{"file_id": "abc123", ...}

# 2. 提取知识
curl -X POST http://localhost:8000/api/v1/fault-tree/extract \
  -H "Content-Type: application/json" \
  -d '{"file_id":"abc123","mode":"multimodal"}'

# 返回：{"triplets":[...], "events":[...], "gates":[...]}

# 3. 生成故障树
curl "http://localhost:8000/api/v1/fault-tree/generate_tree?top_event=系统故障"
```

### 场景 3: 批量处理文件

```python
from pathlib import Path
from app.services.knowledge_extraction_service import KnowledgeExtractionService

service = KnowledgeExtractionService()

# 处理 docs/目录下所有 PDF 文件
for pdf_file in Path("docs").glob("*.pdf"):
    result = service.extract_and_save(str(pdf_file))
    print(f"{pdf_file.name}: {len(result['triplets'])} 个三元组")
```

## ❓ 常见问题

### Q1: 支持哪些文件格式？

**A**: 目前支持：
- ✅ `.txt` - 纯文本
- ✅ `.md` - Markdown
- ✅ `.pdf` - PDF 文档（需要 pdfplumber）
- ✅ `.docx` - Word 文档（需要 python-docx）
- ✅ `.csv` - CSV 表格

### Q2: 提取的准确率如何？

**A**: 
- 常规故障描述：~90%+
- 复杂嵌套逻辑：~80%+
- 模糊/不确定描述：~70%+

可通过优化 prompt 和人工审核提高准确率。

### Q3: 如何处理大文件？

**A**: 
- 小文件（<1MB）：直接处理
- 中等文件（1-10MB）：建议使用 multimodal 模式
- 大文件（>10MB）：建议拆分或使用异步任务队列（待实现）

### Q4: 能否自定义提取规则？

**A**: 可以，修改 `triplet_extractor.py` 中的 `system_prompt`。

例如添加特定领域的故障类型定义。

### Q5: 没有 Neo4j 能用吗？

**A**: 可以！知识抽取功能可以独立使用，只保存为 JSON 文件：

```python
service = KnowledgeExtractionService()
result = service.extract_from_file("test.pdf")
# 不插入数据库，仅保存 JSON
```

### 问题：文件格式不支持

```bash
# 查看支持的格式
python -c "from app.services.knowledge_extraction_service import KnowledgeExtractionService; print('Supported formats: TXT, MD, PDF, DOCX, CSV')"
```

### 问题：提取结果为空

可能原因：
1. 文档中没有故障相关描述
2. LLM 理解错误（调整 prompt）
3. 置信度阈值过高（默认 0.6）

解决方法：
```python
# 降低置信度阈值
service = KnowledgeExtractionService()
service.extractor.confidence_threshold = 0.5  # 降低到 0.5
```

## 📚 进阶使用

### 技巧 1: 自定义 Prompt

编辑 `backend/app/services/triplet_extractor.py`:

```python
self.system_prompt = """
你是专业的航空领域故障分析专家...

特殊要求:
- 只提取与液压系统相关的故障
- 忽略电气系统故障
- 重点关注机械磨损类故障
"""
```

### 技巧 2: 后处理过滤

```python
result = service.extract_from_file("document.pdf")

# 过滤高置信度三元组
high_conf_triples = [
    t for t in result['triplets'] 
    if t['confidence'] > 0.9
]

# 只要特定类型的关系
causal_triples = [
    t for t in result['triplets'] 
    if t['relation'] == 'resultsIn'
]
```

### 技巧 3: 导出为其他格式

```python
import json

# 导出为 CSV
import csv
with open('triplets.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.DictWriter(f, fieldnames=['subject', 'relation', 'object'])
    writer.writeheader()
    for t in result['triplets']:
        writer.writerow({
            'subject': t['subject_name'],
            'relation': t['relation'],
            'object': t['object_name']
        })
```
