# 前端界面风格测试

本测试用于验证项目的设计风格。

## 🎨 测试内容

### 最终确定的设计组合

**组合 C: Naive UI + Tailwind CSS**
- 模拟 Naive UI 的设计风格
- 深色主题 + 毛玻璃效果
- 现代感强烈的渐变色和光晕效果

## 🚀 启动方式

### 方法 1: 双击批处理文件
```bash
# 双击运行
frontend/启动界面测试.bat
```

### 方法 2: 命令行启动
```bash
cd frontend
npm run test:theme
```

## 📌 访问地址

- **测试页面**: http://localhost:3001/test-theme
- **主应用**: http://localhost:3000 (不受影响)
